# Чеклист требований

## ✅ Выполненные требования

### 1. REST API
✅ **CLI использует внешний REST API arXiv**
- Endpoint: `http://export.arxiv.org/api/query`
- Параметры: `search_query`, `start`, `max_results`, `sortBy`, `sortOrder`
- Формат ответа: Atom XML
- Парсинг через `feedparser`

### 2. Обе версии работают
✅ **ver-claude** — полностью реализована, протестирована
❓ **ver-gpt** — реализована параллельно (в другой папке)

### 3. Все команды реализованы
✅ **11 команд согласно ТЗ:**
1. `search` — поиск с фильтрами, пагинацией, датами
2. `download` — скачивание PDF с автоименованием и batch
3. `add` — добавление в библиотеку без скачивания
4. `remove` — удаление из библиотеки
5. `list` — просмотр библиотеки с фильтрами
6. `info` — детальная информация + управление тегами
7. `export` — экспорт BibTeX/CSL с фильтрами
8. `track` — отслеживание версий статей
9. `digest` — дайджесты с группировкой и Markdown
10. `subscribe` — подписки с детекцией новых статей
11. `watch` — заглушка (опционально)
