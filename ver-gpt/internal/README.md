# internal/

Внутренние пакеты проекта (не для внешнего импорта).

Ожидаемые модули:
- `internal/arxiv` — клиент arXiv API + парсинг Atom
- `internal/cli` — команды и флаги
- `internal/config` — конфиг, профили, env, paths
- `internal/digest` — сборка дайджестов
- `internal/export` — BibTeX/CSL
- `internal/storage` — хранение сохранённых запросов/кеша
- `internal/utils` — утилиты
